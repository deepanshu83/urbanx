# UrbanX — Next.js E-Commerce Template

**Full-stack fashion store with Supabase backend**

## Quick Start

```bash
npm install
npm run dev
```

**Website**: http://localhost:3000  
**Admin**: http://localhost:3000/admin  
**Supabase**: Required (see root README.md)

## Tech Stack
- Next.js 16 (App Router)
- React 19 + TypeScript 5
- Tailwind CSS 4
- Supabase (DB + Storage)

## Admin Login
```
urbanx_admin / UrbanX@secure24
```

See root `README.md` for full Hindi/English docs + setup!
